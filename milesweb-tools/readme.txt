== Description ==
MilesWeb Tools is a powerful WordPress plugin designed to enhance security, manage maintenance mode, enforce HTTPS, disable file editing, user logins log, track storage usage, and provide detailed insights into active/inactive themes and plugins.

Features:

- Enable/disable maintenance mode.
- Force HTTPS redirection.
- Disable file editing in the WordPress admin panel.
- Disable XML-RPC to prevent security vulnerabilities.
- Track user logins.
- Display active theme details and inactive theme storage usage.
- Show active/inactive plugins, including their size, and memory usage.
- Identify outdated plugins and themes.

== Installation ==
**Minimum Requirements:**
- WordPress 5.6 or higher
- PHP 7.4 or higher

**Automatic Installation**:
   - This plugin will be automatically installed if you set up a WordPress site on Mileswebâ€™s managed WordPress hosting account.

**Install via WordPress Admin**:
   1. In your WordPress admin panel, navigate to 'Plugins' > 'Add New'.
   2. Search for "MilesWeb Tools" in the search bar.
   3. Click 'Install Now' next to the plugin.
   4. Once the installation is complete, click 'Activate'.
   5. Access the plugin settings via the 'MilesWeb' menu in the WordPress admin panel.

**Manual Installation**:
   1. Download the plugin ZIP file from WordPress.org.
   2. Extract the ZIP file and upload the milesweb-tools folder to the /wp-content/plugins/ directory on your server.
   3. Go to the 'Plugins' menu in your WordPress admin panel and activate the plugin.
   4. Access the plugin settings via the 'MilesWeb' menu in the WordPress admin panel.
